## üöÄ Project Setup

To get a local copy up and running, follow these simple steps.

### Prerequisites

You will need [Node.js](https://nodejs.org/) (which includes npm) installed on your machine.

### Installation

1.  Clone the repo:
    ```sh
    git clone [https://github.com/Bhagvati-Gaikwad/DineFinder.git](https://github.com/Bhagvati-Gaikwad/DineFinder.git)
    ```
2.  Navigate to the project directory:
    ```sh
    cd DineFinder
    ```
3.  Install NPM packages:
    ```sh
    npm install
    ```

### Running the Project

**Compiles and hot-reloads for development:**
```sh
npm run serve
```
Open [http://localhost:8080](http://localhost:8080) (or the port shown in your terminal) to view it in the browser.

**Compiles and minifies for production:**
```sh
npm run build
```

**Lints and fixes files:**
```sh
npm run lint
```
